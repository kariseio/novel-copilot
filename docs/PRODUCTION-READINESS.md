# 프로덕션 준비 명세 — 인터랙션 모드 · 보장 등급 · 측정 체계

> 2026-06-10 기준. 이 문서는 "무엇이 자동이고, 무엇이 사용자와의 인터랙션이고, 무엇이 사용자 전용인가"와
> "엔진이 무엇을 어느 강도로 보장하는가(실측 포함)"를 제품 계약으로 고정한다.

## 1. 인터랙션 모드 매트릭스 (제품 계약)

**원칙(비대칭 불변식의 UX 판):** 캐논(binding ground_truth)을 바꾸는 행위는 전부 ③ 사용자 입력 전용.
시스템 ① 자동은 비구속(narrative/advisory)까지만 쓴다. 그 사이 — 시스템이 제안하고 사용자가 확정하는 것 — 이 ② 인터랙션.

| 기능 | 모드 | 신뢰층 | 비고 |
|---|---|---|---|
| RAG 색인/검색, 줄거리 요약(story_so_far), 계층 롤업 | ① 자동 | narrative | 캐논 불변 |
| LLM Wiki ingest/lint | ① 자동 | narrative | 결정론 코어는 위키를 읽지 않음 |
| 회차 자동추출(신규 엔티티/관계/상태) | ① 자동 | provisional / narrative_inferred | 비가역·terminal 전이는 작가 확정 전 비구속 |
| 가역 상태 전진(소속·등급 등) | ① 자동 (예외) | ground_truth | '추가/전진만' 정책: 가역+단조가드+모순 시 escalation |
| 일관성 게이트 검사·부분 재작성 | ① 자동 | — | 본문만 수정, 캐논 불변 |
| 드리프트 신호(cast_missing/pacing) | ① 자동→② | advisory | 시스템 계산, 대응은 작가 선택 |
| 떡밥 회수 되먹임(미회수 복선→비트 설계 주입 + plant_backlog 경보) | ① 자동→② | advisory | 회수 '우선 고려' 주입 — 슬로우번은 작가 지시로 우회(마감 강제 아님) |
| 보이스 카드(인물별 말투 시그니처 draft 주입) | ① 자동 | 스타일 지침 | 시그니처 편집은 ③(worldgen 생성→작가 수정) |
| 비용 계측(단계별 토큰) | ① 자동 | — | usage_by_stage |
| 월드빌딩 대화 제안(엔티티/관계/설정집) | ② 인터랙션 | 잠정(provisional/ai_unreviewed) | 작가가 그래프/설정집에서 확정 시 캐논 |
| ESCALATED 회차 처리(재시도/지시 수정) | ② 인터랙션 | — | 시스템이 막고, 사용자가 결정 |
| AI 추정 관계(점선) → 캐논 승격 | ② 인터랙션 | narrative_inferred → ground_truth | 승격 행위 자체는 ③ |
| 설정집 promote("캐논으로 박기") | ③ 입력 전용 | → world_rule(SEMANTIC) | advisory 강도 — 아래 §2 |
| 관계 긋기/종료(ground_truth 엣지) | ③ 입력 전용 | ground_truth | 시간선 모순 게이트 통과 시만 |
| 엔티티 추가, 설정집 편집, 빨간펜 지시 | ③ 입력 전용 | — | |
| 시드/세계 생성 파라미터, eff_from, allow_state_reversal | ③ 입력 전용 | — | |

## 2. 보장 등급표 (실측 게이트 유효성 포함)

측정: `tools/measure_gates.py` — 결정론 골든셋(위반 5종 + 하드네거티브) 대비 confusion matrix.
최신 실측(2026-06-10, **n=30/유형 + 하드네거티브 84**, gpt-4.1, `tools/reports/gate_validity.json`, exit 0=배포게이트 통과):

| 축 | 주입(프롬프트) | 게이트 | 등급 | 실측 Recall (CI95) | 실측 하드네거티브 FPR |
|---|---|---|---|---|---|
| 생애주기 terminal(사망 등) 현재행동 | O (중대 상태만) | **hard** | quasi | **1.00** [0.886, 1.0] | **0/84, CI상한 0.044** (회상/환영/언급 전부 통과) |
| 범주형 속성(눈색 등) | O | **hard** | quasi | 0.90 [0.744, 0.965] | 0/84 |
| 단조 수치(등급 등) | O | **hard** | quasi | 1.00 [0.886, 1.0] | 0/84 |
| 관계 상충(동맹↔적대 등 선언된 conflicts_with) | O | **hard** | quasi | **1.00** [0.886, 1.0] | 0/84 |
| 세계규칙(world_rule) | O | **advisory — 차단 없음** | semantic | 1.00 (탐지) | — (재작성/ESCALATED 강제하지 않음. '캐논으로 박기'=추적·주입 보장이지 차단 보장 아님) |
| SSOT 내부 모순(그래프/타임라인) | — | **hard** | deterministic | LLM 무관(코드) | — |
| pov(인식/믿음) 엣지 | X (객관 캐논 비주입) | 없음 | — | 관점 사실은 의도적으로 비구속 |
| 보이스/페이싱 | O (보이스 카드/비트 제약) | 없음 | 프록시 advisory | 측정 도구만(§3) |

추출 신뢰 장치: **증거 스팬 강제**(값·flag·관계 단정은 본문 인용이 실재해야 클레임 인정 — 환각의 결정론 차단),
하드네거티브 few-shot(회상/환영/언급 분류), '기타' 탈출구(통제어휘 밖 → escalation).
운영 임계: 유형별 Recall ≥ 0.8 AND 하드네거티브 FPR ≤ 0.05 미달 시 해당 유형 advisory 강등(데이터 주도 재측정).

## 3. 측정 체계 (이 제품이 좋아지는지 아는 방법)

| 측정 | 도구 | 상태 |
|---|---|---|
| 게이트 유효성(Recall/Precision/FPR + Wilson CI) | `tools/measure_gates.py` | **가동, 실측 완료** |
| 단위경제(단계별 토큰: plan/draft/check/rewrite/wiki/summarize/propose → 일관성 오버헤드율) | `ChapterRecord.usage_by_stage` | **계측 가동** |
| 재미 프록시(대사비율·훅·복선회수율·사건밀도) | `tools/measure_fun_proxies.py` | 가동(인간 보정 전 프록시) |
| 블라인드 선호(엔진 on/off A/B, 실독자) | 프로토콜 §5 | **미실행 — 사람 필요** |
| Ablation(풀엔진 vs 롱컨텍스트 4군) | 측정 하네스 재활용 | 미실행(비용 승인 필요) |

## 4. 운영 계약
- **단일 uvicorn 워커 전용**(락·세션캐시가 프로세스-로컬). 멀티워커 전환 전 lock 권위의 DB 이관 필수.
- 영속: 핫패스 JSON(수십 KB) + **회차별 임베딩 샤드**(`{pid}.rag.{ch}.json`, 지문 변경 샤드만 재기록 = finalize 당 O(1회차)).
  구형 인라인/단일 사이드카 자동 마이그레이션. 검색은 인스턴스 캐시 색인(BM25·정규화 행렬, finalize 시에만 재구축).
- 게이트 유효성은 추출 모델 교체 시마다 재측정(`measure_gates.py`)이 배포 게이트(미달 유형은 advisory 강등).
- 재미 프록시: 훅은 반복(재탕) 페널티 보정 포함. payoff_recovery 는 생성측 되먹임(§1 떡밥 회수)과 한 고리.
